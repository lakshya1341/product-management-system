# 🛒 Product Management System

A role-based product management system built using React, TypeScript, and Redux. This project simulates real-world enterprise workflows with role-based access and approval systems.

---

## 🚀 Features

- 🔐 Role-Based Access Control (Admin, Manager, User)
- ✅ Multi-level approval workflow for product actions
- 📝 Create, Update, Delete (CRUD) product functionality
- 📊 Dynamic forms with validation
- 🔄 Redux state management
- 🌐 API integration and error handling

---

## 🛠 Tech Stack

- React.js
- TypeScript
- Redux
- Tailwind CSS

---

## 📌 Project Description

This project is designed to mimic a real-world product/inventory management system used in companies. Users can perform actions based on their roles, and certain actions require approval before being finalized.

It demonstrates:
- Scalable frontend architecture
- State management using Redux
- Role-based authorization logic
- Debugging and handling real-world edge cases
